# 🌿 Hanoi Seasons Gallery

An interactive travel-inspired gallery that explores the four seasons of Hanoi through motion, storytelling, and bilingual content. Built with React, TypeScript, Motion, Tailwind CSS, and Vite, each season opens into an editorial-style page with animated hero sections, seasonal cards, smooth language switching, and floating visual interactions.

## ✨ Technologies

- `React`
- `TypeScript`
- `Motion`
- `Tailwind CSS`
- `Vite`
- `i18next`
- `React Router`

## 🚀 Features

- Interactive season titles with hover-based image previews
- Mouse-tracked floating images with parallax motion
- Dedicated pages for Spring, Summer, Autumn, and Winter
- Bilingual support for English and Vietnamese
- Smooth language transition animations
- Animated HeroBand intro for each season
- Scroll-triggered section cards with seasonal content
- Responsive layout with mobile-friendly adjustments
- Back-to-top button and thank-you popup near the end of each page

## 📍 The Process

I wanted to create a visual storytelling experience inspired by Hanoi’s seasonal atmosphere. Instead of building a static gallery, I focused on making the interface feel calm, poetic, and alive.

The homepage started with four simple season titles, then evolved into an interactive gallery where hovering over each season reveals floating scenes that respond to mouse movement. Each season page was designed like a small editorial article, with animated hero text, seasonal gradients, icons, and cards describing weather, scenery, places, and food.

Later, I added bilingual support so users can switch between English and Vietnamese instantly. The content is structured separately for each season, making it easier to maintain and expand. Motion animations were used throughout the project to create soft transitions, scroll reveals, and cinematic intro effects.

The project is still being refined for mobile, but the main goal is to capture the feeling of Hanoi through design, motion, and atmosphere.

## 🚦 Running the Project

1. Clone the repository
2. Install dependencies: npm install
3. Run development server: npm run dev
4. Open the local server in your browser

## 🪄 Notes

This project is best experienced on desktop because the homepage relies on hover interactions and mouse-tracked motion effects. On mobile, the layout is adjusted for readability, and some hover-based visuals may be reduced or hidden.
